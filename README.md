# ADD File
```
    git init
    git add . //add All file 
    git status
    git commit -m "Test" 
    git branch -M main
    git remote add origin https://github.com/nuntipatcpe/test.git
    git push -u origin main
```

# Branch
```
    git branch
    git branch -M main 
    git branch -c new_branch //create branch
    git branch -d newBranch 
    git branch -D newBranch //force delete
```
